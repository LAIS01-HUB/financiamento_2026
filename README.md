# Simulador de Financiamento
---

##  Sobre o projeto

Devido às altas taxas de juros, financiar bens como carros, motos, imóveis ou até eletrodomésticos pode sair muito mais caro do que o esperado.

Este aplicativo foi desenvolvido para ajudar o usuário a visualizar, de forma clara, o impacto dos juros em um financiamento antes de tomar uma decisão.

---

##  Objetivo

Criar um aplicativo que permita simular um financiamento a partir das seguintes informações:

- Valor do bem  
- Número de parcelas  
- Taxa de juros mensal  
- Taxas adicionais  

E apresentar:

- Valor de cada parcela  
- Montante total pago ao final do financiamento  

---

##  Funcionalidades

✔ Inserção de dados do financiamento  
✔ Cálculo automático das parcelas  
✔ Cálculo do valor total pago  
✔ Inclusão de taxas adicionais  
✔ Interface simples e intuitiva
